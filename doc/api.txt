URL Path:
用户
1. 请求验证码
Path: user/getPhoneCode
Param: phone*
Return: json {result: "{code}"}
 code: 	0	成功
 		1	手机号码必填
		2	手机号码格式错误
		3 	手机号码已存在
		1001 服务器错误

2. 验证验证码
Path: user/checkPhoneCode
Param: code*
Return: json {result: "code"}
 code: 	0	成功
 		1	验证码必填
		2	验证码错误
		6 	Session过期
