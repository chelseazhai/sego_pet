http://{server_path}/{path}

server_path: www.segopet.com/segopet

Comment: * 为必填

URL Path:
用户
1. 请求验证码
Path: user/getPhoneCode
Param: phone*
Return: json {result: "{code}"}
 code: 	0	成功
 		1	手机号码必填
		2	手机号码格式错误
		3 	手机号码已存在
		1001 服务器错误

2. 验证验证码
Path: user/checkPhoneCode
Param: code*
Return: json {result: "code"}
 code: 	0	成功
 		1	验证码必填
		2	验证码错误
		6 	Session过期
		
3. 完成注册
Path: user/regUser
Param: 	password*
		password1*
		deviceId
		nickname
Return: json 	{
					result : "code"
					userId : string, 
					username : string, 
					userkey : string
				}
 code: 	0	成功
 		1	失败
 		6	Session过期
 		7	手机号不同

4. 登录
Path: user/login
Param: 	loginName*
		loginPwd*
		brand
		model
		release
		sdk
		width
		height
Return: json 	{
					result : "code", 
					userId : string, 
					username : string, 
					userkey : string,
					nickname : string
				}
 code: 	0	成功
 		1	失败
 		
 宠物信息：
 1. 修改创建宠物信息
 Path: 	petinfo/modify
 Param: username*
		petid		- 空则为创建
		nickname
		sex
		breed
		age
		height
		weight
		district
		placeoftengo
Return: json 	{
					result : "code",
					id : string
				}
 code: 	0	成功
		1 	修改失败
		3	已经有宠物
		4	创建失败
		
2. 获取宠物列表
Path: 	petinfo/getpets
Param: 	username*
Return: json	{
					result : "code",
					list : [array of PetInfo]
				}
		PetInfo structure:
				{
					petid : string,
					nickname : string,
					sex : string,
					weight : string,
					chuanganid : string,
					ownerid : string,
					avatar : string,
					breed : string,
					age : string,
					height : string,
					district : string,
					placeoftengo : string,
					deviceid : string,
					score : string
				}
 code: 	0	成功
				
3. 获取宠物详细信息
Path:	petinfo/getpetdetail
Param:	petid*
Return: json PetInfo（同上）,多一个 result : "code"
 code: 	0	成功
